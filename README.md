# 🎬 Royal Films - Sidebar Gallery Project

A responsive **HTML + CSS web project** featuring a modern sidebar navigation and image gallery.  
Built for practice and portfolio showcase using pure frontend technologies.

---

## 🚀 Features

* 🎨 Responsive sidebar navigation menu  
* 📸 Image gallery with 20+ photos  
* 📱 Mobile-friendly layout  
* 🎭 Smooth hover and transition effects  
* 🔗 Font Awesome icons integration  
* 🌈 Background image-based UI design  

---

## 🛠️ Tech Stack

* HTML5  
* CSS3  
* Font Awesome Icons  
* Google Fonts (Poppins)  

---

## 📂 Project Structure
```
PROJECT Mini HTML+CSS/
│
├── index.html        # Main homepage (Sidebar UI)
├── style.css         # Styling for homepage + sidebar
│
├── gallery.html        # Image gallery page
├── gallery.css         # Styling for gallery grid
│
├── camera.jpg          # Background / main image asset
│
└── gallery/            # Image folder (20 images inside)
```

---

## ⚙️ How to Run

Clone the repository:
```
git clone https://github.com/your-username/royal-films-gallery.git
```
Open project folder:
```
cd royal-films-gallery
```
Run project:
```
Open index.html in your browser
```

---

## 💻 Usage

* Open `index.html` to view homepage with sidebar  
* Click on **Gallery** to view image gallery  
* Navigate using sidebar menu icons  
* Fully responsive on mobile devices
